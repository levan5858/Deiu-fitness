# YouTube Video IDs for Andrei Deiu

Replace `YOUTUBE_VIDEO_ID_1` through `YOUTUBE_VIDEO_ID_10` in the HTML files with actual video IDs from Andrei Deiu's YouTube channel.

## How to find video IDs:

1. Go to Andrei Deiu's YouTube channel: https://www.youtube.com/@AndreiDeiu
2. Open any video
3. The video ID is in the URL: `https://www.youtube.com/watch?v=VIDEO_ID_HERE`
4. Replace the placeholder IDs in:
   - `index.html` (1 video)
   - `media.html` (10 videos in gallery + 1 featured video)

## Example:
- If video URL is: `https://www.youtube.com/watch?v=abc123xyz`
- Use in embed: `https://www.youtube.com/embed/abc123xyz`
